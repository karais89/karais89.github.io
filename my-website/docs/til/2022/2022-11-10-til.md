- deta 배포 진행 플로우
    1. https://www.deta.sh 회원 가입
    2. `curl -fsSL https://get.deta.dev/cli.sh | sh` 명령어 사용 후 deta 설치
    3. deta 설치 경로 환경 변수 설정 혹은 해당 경로 기억
    4. fast api 프로젝트 준비
    5. fast api 프로젝트에서 아래 date 명령어 입력 후 배포
        ```sh
        deta new # 로그인
        deta new # 최초 프로젝트 생성시
        deta deploy # 이후 프로젝트 변경시       
        ```
